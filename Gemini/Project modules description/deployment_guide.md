# Инструкция по деплою CALIDAD ERP

Для того чтобы ваша "пустышка" ожила по реальному адресу в интернете, выполните следующие шаги:

## 1. Подготовка в консоли Firebase
1. Перейдите в [Firebase Console](https://console.firebase.google.com/).
2. Нажмите **"Add project"** и создайте проект с именем `calidad-erp-test`.
3. В левом меню выберите **Build -> Hosting** и нажмите **"Get started"**.

## 2. Локальная установка (на вашем компьютере)
Если у вас еще не установлены инструменты Firebase, выполните в терминале:
```bash
npm install -g firebase-tools
```

## 3. Авторизация и сборка
В корневой папке проекта выполните:
```bash
# 1. Зайти в аккаунт Google
firebase login

# 2. Собрать проект (Vite создаст папку dist)
npm run build

# 3. Отправить в облако
firebase deploy
```

## 4. Переменные окружения в CI/CD (Опционально)
Если вы будете использовать GitHub Actions для авто-деплоя, не забудьте добавить `VITE_SUPABASE_URL` и `VITE_SUPABASE_ANON_KEY` в **GitHub Secrets**, чтобы они попали в билд.

---
**Адрес вашего приложения появится в консоли после выполнения `firebase deploy`.**
