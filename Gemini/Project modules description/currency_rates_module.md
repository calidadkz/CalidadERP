# Модуль: Курсы валют (Currency Rates)

Финансовый контроллер системы, отвечающий за актуальность стоимостных показателей. Модуль управляет базовыми курсами валют относительно тенге (KZT) и инициирует глобальные пересчеты цен в каталоге.

---

## 1. Основной экран: Сетка курсов

Визуальное представление текущих котировок системы.

### Шапка управления (Header)
*   **Заголовок:** «Курсы валют» с иконкой `RefreshCcw` цвета индиго.
*   **Центр управления:**
    *   **Сохранить:** Появляется только при изменении значений в полях.
    *   **Пересчитать цены (Calculator):** Акцентная кнопка (становится `bg-emerald-600` с пульсацией после сохранения курсов). Запускает фоновый процесс обновления цен всех товаров и шаблонов.

### Сетка валют (Cards):
Крупные карточки для USD, EUR, CNY, RUB.
*   **Индикация:** Если значение в поле отличается от сохраненного в БД, карточка подсвечивается синим (`bg-blue-50`) и выводится предупреждение «Сохраните курс».
*   **Точность:** Поля ввода поддерживают дробные значения через точку или запятую.

---

## 2. Бизнес-логика: Глобальная синхронизация

Модуль является триггером для `PricingService`:
*   **Массовый пересчет:** При запуске система последовательно проходит по всей номенклатуре и пересчитывает `Sales Price` на основе новой стоимости валюты закупа и параметров примененных профилей.
*   **Обратная связь:** Во время пересчета отображается оверлей с прогресс-баром и детализацией: сколько товаров и шаблонов было успешно обновлено.

---

## 3. Аналитика: Кросс-курсы и Справочники

Нижний блок в темном исполнении (`bg-slate-900`):
*   **Кросс-курсы:** Автоматический расчет пар USD/CNY, EUR/USD и CNY/RUB. Помогает отделу закупа оценивать выгодность конвертации.
*   **История:** Информационная плашка с временем последнего успешного глобального пересчета цен.

---

## 4. Визуальные стандарты UI
*   **Состояния:** 
    *   `ring-emerald-500/20`: Для активной кнопки пересчета.
    *   `animate-spin-slow`: Для иконок обновления в измененных карточках.
*   **Модальные окна:** Использование `backdrop-blur-md` для окна подтверждения глобального пересчета, чтобы акцентировать внимание на важности действия.
