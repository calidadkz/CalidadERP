
# Модуль: Ценообразование (Pricing / Unit Economics)

Инструмент автоматизации финансового планирования, позволяющий системе самостоятельно рассчитывать продажные цены на основе целевой рентабельности и констант логистических затрат. Модуль исключает ошибки ручного ввода наценок и гарантирует плановую маржинальность каждой сделки.

---

## 1. Основной экран: Управление профилями

Центральный реестр экономических моделей (PricingProfile).

### Шапка управления (Header)
*   **Заголовок:** «Ценообразование» с иконкой `Calculator` цвета индиго.
*   **Инструменты:**
    *   `Импорт/Экспорт`: Синхронизация профилей через CSV (удобно для массового обновления тарифов логистики).
    *   `Новый профиль`: Акцентная синяя кнопка (`bg-blue-600`), открывающая Wizard-редактор.

### Сетка профилей (Grid)
Карточки с краткими характеристиками модели:
*   **Название / Поставщик:** Идентификация профиля (напр., «Станки JK - Море»).
*   **Целевая рентабельность:** Крупный индикатор в углу карточки (напр., `25%`).
*   **Действия:** Быстрый запуск **«Глобального пересчета»** (`RefreshCw`) для обновления цен всех товаров, привязанных к данному профилю.

---

## 2. Бизнес-логика: "Умный расчет" (Smart Pricing)

Модуль реализует алгоритм **обратного вычисления цены (Reverse Pricing)**:
1.  **Direct Costs:** Суммируются все прямые расходы (Закуп + Логистика + Таможня + Сервис).
2.  **Tax & Margin Denominator:** Вычисляется коэффициент, учитывающий, что налоги (НДС 16%) и бонусы менеджеров считаются от *финальной* (еще не известной) цены.
3.  **Final Price:** `Прямые расходы / (1 - %НДС_эфф - %Бонус - (%Целевая_Маржа / (1 - %КПН)))`.
4.  **Результат:** Цена, которая гарантирует получение именно того процента чистой прибыли, который указан в поле профиля.

## 3. Редактор профиля (Wizard Modal)

Полноэкранное окно (`max-w-5xl`), разделенное на три логические колонки:

### Колонки параметров:
1.  **Логистика (Синяя зона):** 
    *   Тариф за м³ из Китая ($), объем контейнера и стоимость партии (KZT). 
    *   Система использует эти данные для распределения затрат на единицу товара пропорционально его объему.
2.  **Налоги и Таможня (Оранжевая зона):** 
    *   Ставки НДС (16%) и КПН (20%), затраты на брокера и таможенные сборы.
3.  **Сервис и Цель (Изумрудная зона):** 
    *   Стоимость ПНР (пусконаладки), доставки до клиента и бонус отдела продаж.
    *   **Центральный виджет — «Рентабельность»:** Главный драйвер математики профиля.

---

## 4. Визуальные стандарты UI
*   **Стиль «Dashboard»:** Крупные поля ввода с иконками (`Truck`, `Landmark`, `Target`).
*   **Интерактив:** 
    *   При изменении любого параметра в редакторе, нижняя плашка «Итого» должна пульсировать, сигнализируя об изменении расчетной модели.
    *   Использование `ring-emerald-500/20` для полей целевой рентабельности.
*   **Типографика:** Использование `font-mono` для всех числовых коэффициентов и сумм в KZT.
