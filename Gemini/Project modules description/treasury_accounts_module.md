
# Модуль: Наши счета (Treasury & FIFO Analytics)

## 1. Структура данных и Функционал
Управление ликвидностью и валютными резервами, реализованное в `FinanceAccounts.tsx`.

### Счета и Балансы
*   **Сетка счетов:** Интерактивные карточки. При клике на счет правая панель FIFO фильтрует лоты только для выбранного банка.
*   **Внутренние операции:** 
    *   **Перевод:** Между счетами одной валюты с учетом комиссии.
    *   **Обмен:** Конвертация (напр. KZT -> CNY), создающая новый входящий лот FIFO с вычисленным курсом закупа.

### FIFO Аналитика
*   **Стеки (Lots):** Отображают историю приобретения валюты. Каждая запись содержит `amountRemaining` — сколько еще "дешевой" или "дорогой" валюты осталось.

## 2. Дизайн и Интерфейс (UI/UX)

### Dashboard счетов
*   **Карточки:** Белый фон, жирный шрифт баланса, иконка `Landmark`. Активный счет выделяется синей рамкой и тенью.
*   **Валютные бейджи:** В углу каждой карточки — код валюты на сером фоне (USD, CNY, RUB).

### Панель операций (Dark & Light Mix)
*   **Форма:** Использование вкладок (Tabs) для переключения Перевод/Обмен.
*   **Инпуты:** Сумма списания подсвечивается красным фоном (сигнал расхода), сумма получения — зеленым.
*   **Кнопка «Выполнить»:** Крупная темная кнопка (`bg-slate-900`) с эффектом нажатия `active:scale-95`.

### FIFO Визуализация
*   **Прогресс-бары:** Горизонтальные линии под каждым лотом показывают процент использования. Синий цвет линии — активный остаток.
