# Модуль: Категории (Categories)

## 1. Структура данных и Функционал
Справочник для классификации номенклатуры. Служит фильтром для подбора опций и организации каталога.

### Поля (ProductCategory)
*   **id:** Уникальный системный идентификатор (string).
*   **name:** Название категории (напр., "Лазерные станки CO2").
*   **type:** Привязка к типу продукта (`ProductType`).

### Логика работы
*   **Импорт/Экспорт (CSV):**
    *   Формат: `Название;Тип`.
    *   Типы должны строго соответствовать значениям: "Станок", "Запчасть", "Услуга".
    *   При импорте система игнорирует дубликаты (совпадение названия и типа).
*   **Права доступа (Permissions):** 
    *   `create`: Разрешает добавление новых категорий и импорт.
    *   `delete`: Разрешает архивацию в корзину.
*   **Удаление:** Реализовано через механизм `moveToTrash`.

---

## 2. Дизайн и Интерфейс (UI/UX)

### Основной экран
*   **Инструменты (Header):** Кнопки экспорта и импорта в правой части заголовка. Использование стандартных иконок `Download` и `Upload`.
*   **Компоновка:** Центрированный узкий макет (`max-w-4xl`) для удобства чтения списка.
*   **Навигация (Tabs):** Кнопки-переключатели в стиле "Segmented Control" с акцентными цветами (Синий/Оранжевый/Индиго).

### Список категорий
*   **Таблица:** В стиле "Enterprise" с большими отступами.
*   **Удаление:** Используется **Inline Confirmation** для предотвращения случайных нажатий.
