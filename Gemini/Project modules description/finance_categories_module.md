# Модуль: Статьи ДДС (Finance Categories / Cash Flow Items)

Фундаментальный справочник для управленческого учета. Модуль обеспечивает аналитическую разбивку всех денежных потоков компании, позволяя классифицировать движения средств в Календаре (IPP) и Выписках (IP).

---

## 1. Основной экран: Управление статьями

Центрированный интерфейс для организации структуры доходов и расходов.

### Шапка управления (Header)
*   **Заголовок:** «Статьи ДДС» с иконкой `Tag` цвета индиго.
*   **Инструменты:**
    *   `Экспорт/Импорт`: Кнопки для массовой загрузки дерева статей через CSV (формат: `Название;Направление`).
    *   `Новая статья`: Inline-поле ввода в верхней части списка для быстрого добавления.

### Навигация (Tabs):
Переключатель направлений в стиле "Segmented Control":
*   **Расходы:** Статьи с красным индикатором (напр., «Оплата за товар», «Аренда»).
*   **Доходы:** Статьи с зеленым индикатором (напр., «Выручка от продаж», «Возврат депозита»).

---

## 2. Бизнес-логика и Интеграция

*   **Связь с транзакциями:** Каждая плановая (IPP) или фактическая (IP) оплата обязана иметь привязку к статье ДДС для корректного формирования отчета о движении денежных средств.
*   **Удаление (Recycle Bin):** Статьи не удаляются физически, а перемещаются в **Корзину**. Это сохраняет историческую целостность финансовых документов, где эта статья была использована.
*   **Валидация:** При импорте система автоматически сопоставляет текстовые значения «Доход»/«Расход» с внутренним перечислением системы.

---

## 3. Визуальные стандарты UI
*   **Цветовые коды:** 
    *   `bg-red-500` для категорий расходов.
    *   `bg-emerald-500` для категорий доходов.
*   **Типографика:** Использование `font-mono` для отображения коротких системных ID статей.
*   **Интерактивность:** Использование `group-hover` для кнопок удаления, чтобы не перегружать таблицу лишними элементами при чтении.
