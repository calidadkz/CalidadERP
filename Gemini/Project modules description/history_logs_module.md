# Модуль: История (Логи / Audit Log)

Система тотального аудита, обеспечивающая прозрачность действий пользователей и целостность бизнес-процессов. Модуль фиксирует каждое изменение состояния базы данных.

---

## 1. Основной экран: Журнал операций

Централизованный реестр всех системных событий.

### Шапка управления (Header)
*   **Заголовок:** «История операций» с иконкой `History` цвета индиго.
*   **Инструменты:**
    *   **Живой поиск:** Фильтрация по Business ID (напр. по номеру заказа или артикулу товара).
    *   **Фильтр действий:** Быстрый отбор по типу (`Создание`, `Удаление`, `Транзакция`).

### Таблица логов:
*   **Дата / Время:** Точный таймштамп события (формат `DD.MM.YYYY HH:mm:ss`).
*   **Действие:** Цветные бейджи:
    *   `Create` (Зеленый): Новые объекты.
    *   `Update` (Синий): Изменения существующих данных.
    *   `Delete` (Красный): Перемещение в корзину.
    *   `Post` (Фиолетовый): Проведение финансовых или складских документов.
*   **Объект / ID:** Название сущности (напр. "Товар") и его короткий идентификатор.
*   **Описание:** Детализация изменения (напр. "Изменение цены закупа" или "Сторно накладной").
*   **Автор:** Имя сотрудника, совершившего действие.

---

## 2. Бизнес-логика и Автоматизация

*   **Trigger-based Logging:** Логи не создаются вручную. Вызов функции `addLog` интегрирован во все ключевые хуки состояния (`useOrderState`, `useInventoryState`, `useFinanceState`).
*   **Неизменяемость:** Записи в журнале истории доступны только для чтения. Система не предоставляет инструментов для редактирования или удаления логов.
*   **Связность:** Позволяет отследить жизненный цикл объекта от момента создания до перемещения в корзину или окончательного удаления.

---

## 3. Визуальные стандарты UI
*   **Типографика:** Использование `font-mono` для временных меток и ID для исключения двусмысленности при чтении.
*   **Интерактивность:** Подсветка строк при наведении (`hover:bg-gray-50`) для удобства сканирования длинных списков.
*   **Пустые состояния:** При отсутствии записей отображается иконка `Activity` с низкой прозрачностью.
